# 🚑 CareSphere - Smart Emergency Healthcare Locator

CareSphere is a full-stack web application that helps users find nearby hospitals during emergency situations and view available medical facilities such as ICU beds, emergency services, and more.

---

## 🌟 Features

* 📍 Find nearby hospitals using location-based services
* 🏥 View hospital details and available facilities
* ⚡ Fast and responsive UI for emergency situations
* 🔐 User authentication system
* 📊 Backend API for managing hospital data

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* HTML5, CSS3
* JavaScript

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB (MongoDB Atlas)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/CareSphere.git
cd CareSphere
```

---

### 2️⃣ Install Dependencies

#### Backend:

```
cd backend
npm install
```

#### Frontend:

```
cd frontend
npm install
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file inside the backend folder and add:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

---

### 4️⃣ Run the Application

#### Start Backend:

```
cd backend
npm start
```

#### Start Frontend:

```
cd frontend
npm start
```
